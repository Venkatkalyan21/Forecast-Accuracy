📏
MAE — Mean Absolute Error

Average size of the errors, ignoring direction. Easy to understand: if MAE = 2.5°C, the model is typically off by 2.5 degrees.

%
MAPE — Mean Abs. Percentage Error

Same as MAE but as a percentage. Great for comparing models on different datasets. <10% = excellent, 10–20% = good, >20% = needs work.

⚡
RMSE — Root Mean Squared Error

Punishes big outlier errors more than MAE. If RMSE is much higher than MAE, the model made a few very bad predictions.
